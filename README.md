# Savein

Aplikasi manajemen keuangan pribadi berbasis web.

## Fitur
- Manajemen pemasukan dan pengeluaran
- Statistik keuangan
- UI modern

## Instalasi
1. Clone repo ini
2. Install dependencies dengan `npm install` di direktori frontend
3. Jalankan aplikasi dengan `npm start`
